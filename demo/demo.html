About

Devanagari Text Field is small open source JavaScript plugin which enables users to type Devanagari characters in HTML text inputs and textareas using standard Latin script keyboards. The default key mappings are designed to reflect the sound of each Devanagari character rather than attempt to mimic the layout of InScript keyboards. It is hoped that in future versions there will be support for adding custom key mappings.

The project is still in development and is largely untested. However, most of the functionality is there and you can try it out in the text fields below.

Usage

The devangariTextField wrapper is exposed as a global function or as an AMD module. To enable Devanagari script to be typed into a text element simply pass the id of the target element as the first argument. Additional configuration can be achieved by passing your desired settings as an optional second argument as demonstrated below.

Default Settings

devanagariTextField( 'default-tb' );

Auto Add Virama 

devanagariTextField( 'no-auto-virama-tb', {
    autoAddVirama: false
});

Auto Remove Virama 

devanagariTextField( 'no-auto-remove-tb', {
    autoRemoveVirama: false
});

Characters

Almost all the available Devanagari UTF-8 characters can be typed. Unfortunately there are very few fonts which support every character. This is why you will occasionally see blank boxes next to some of the more obscure characters.
