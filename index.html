About

Devanagari Text Field is small open source JavaScript plugin which enables users to type Devanagari characters into HTML text inputs and textareas using standard Roman script keyboards. The default key mappings are designed to reflect the sound of each Devanagari character rather than attempt to mimic the layout of InScript keyboards. It is hoped that in future versions there will be support for adding custom key mappings.

Installation

As a global variable: 


              <!DOCTYPE html>
              <html>
              <head></head>
              <body>
              <input type='text' id='myTextElement'/>
              <script src='devanagariTextField.js'></script>
              <script src='scripts.js'></script>
              </body>
              </html>
          

            // scripts.js
            (function(devanagariTextField){
            devanagariTextField('myTextElement');
            }(devanagariTextField));

Using an AMD loader:


            // scripts.js
            require(['devanagariTextField'], function(devanagariTextField) {
            devanagariTextField('myTextElement');
            });

Characters

Currently there is support for 125 of the 128 characters specified in the Unicode Consortium's chart for standard Devanagari characters. You can review these characters and their default key bindings in the table below.

Unfortunately, not all fonts have support for every Devanagari character. Thus, some of the characters below may appear as a blank box, a box with a cross inside or, in some cases, they may not appear at all. You can change the font used to display the Devanagari characters by clicking on the settings button in the top right cornor of the panel below and choosing a different font family. The fonts used are all available from Google Fonts.
